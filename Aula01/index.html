O que é Máquina Virtual?

É um programa que carrega e executa os aplicativos Java, convertendo os bytecodes em códido executável de máquina.

Por esse motivo, qualquer programa Java pode funcionar em qualquer plataforma, desde que tenha uma versão da JVM.

Como funciona a Máquina Virtual Java?

Características

  • Orientada a Objetos
  • Portável - write once, run anywhere
  • Vários recursos - APIs, Bibliotecas, etc
  • Similar a C/C++
  • Desalocação de memória automática

Orientação a Objetos

É um paradigma de programação baseado na composição e interação entre unidades chamados objetos.

Esse modelo visa os seguintes pontos:

  • Organização
  • Desempenho
  • Sustentabilidade
  • Reutilização

Ela visa um pensamento o mais próximo possível da vida real. Ao programar orientado a objetos não teremos apenas funções que são chamadas uma seguida da outra, mas sim objetosque possuem estadoe comportamento.

Orientação a Objetos

Classes e Objetos

Uma classe é uma estrutura que abstrai um conjunto de objetos com características similares. Uma classe define o comportamento de seus objetos - através de métodos - e os estados possíveis destes objetos - através de atributos.

Em outras palavras, uma classe descreve os serviços oferecidos por seus objetos e quais informações eles podem armazenar.

Um objeto, na vida real, é qualquer coisa que podemos dar um nome. Aqui é uma instância (ou seja, um exemplar) de uma classe.

Um objeto é capaz de armazenar estados através de seus atributos e reagir a mensagens enviadas a ele, assim como se relacionar e enviar mensagens a outros objetos.
