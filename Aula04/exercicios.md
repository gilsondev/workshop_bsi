# Dia 02 - Exercícios

1. Uma empresa solicita o desenvolvimento de um sistema para cadastro de funcionários. Essa empresa possui os seguintes cargos: _Gerente_, _SubGerente_, _Coordenador_ e _Atendente_. Todos os funcionários possui os seguintes dados:

    - Nome;
    - CPF;
    - Idade;
    - Salário;

Existem algumas informações que diferem de cada cargo. Os gerentes tem a opção de receber benefícios mensais no valor de R$ 300,00 por mês, os sub-gerentes no valor de R$ 100,00. Os atendentes recebem uma bonificação a partir de uma porcentagem no valor total de vendas de 10%. Por exemplo, se um atendente vendeu R$ 100,00 ele recebe R$ 10,00 de bonificação.

O sistema deve dar as seguintes opções na tela:

    1. Inserir Funcionário;
    2. Editar funcionário;
    3. Remover funcionário;
    2. Registrar bonificação.

    Digite a opção desejada:


Dicas:

    1. Use arrays para armazenar e acessar os funcionários cadastrados;
    2. Use todos os princípios de Orientação a Objetos ministrados na aula.
